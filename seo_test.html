SEO Meta Tags Test

Use browser developer tools to inspect the meta tags on tool detail pages.

Test URLs:

  • Midjourney Tool
  • AITwo Tool
  • Vibe3D Tool
  • RoomGPT Tool

How to Test:

  1. Open any of the test URLs above
  2. Right-click and select "Inspect" or "View Page Source"
  3. Check the <title> tag - should be unique for each tool
  4. Check the meta description - should be unique and descriptive
  5. Check canonical URL - should point to the specific tool page
  6. Check Open Graph tags (og:title, og:description, og:url)
  7. Check Twitter Card tags (twitter:title, twitter:description)

Expected Results:

  • Title: "Tool Name - Tool Description | Arch AI Tool"
  • Description: "Tool Name: Detailed description. Key features: feature1, feature2, feature3. Pricing info. Read reviews and explore this AI architecture tool."
  • Canonical URL: "https://archaitool.com/tool/tool-id"
